# SOLPONGE 프로젝트 JPA + API 리펙토링
팀 프로젝트로 진행한 solpongeJPA( https://github.com/Yongtapark/jpaSolponge )를 리펙토링한 개인프로젝트입니다.
## 개발 동기
기존의 개인 프로젝트를 마치고, 이후에 공부를 진행하면서 RESTful API와 최적화에 관심이 생겨 진행하게 되었습니다.

### 사전 요구 사항
Java 11
SpringBoot
Spring Data JPA
QueryDsl
H2 DB

##### 4. 데이터 변경 추적
